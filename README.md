# 5hell_tooling
Some tools I use w 5hell to ensure consistency

## 5hell_tests.src
Place this file in the same directory 5hell's source is in, then build the file to any name you want.
Same goes for VS Code, just place it in the folder, and hit Build file from context.

Tests 5hell's functionalities, currently built against 4.3.9, might need updating in the future depending on features added/changes made
